- `Genre` @@map("genre")
- `MovieGenre` @@map("movie_genre")

- `MoviePerson` @@map("movie_person")
- `Company` @@map("company")

- `MoviePersonRole`

## Enums

Stored values:
- `director`
- `writer`
- `producer`
- `cast`

### MovieCompanyRole
Database enum: `movie_company_role`

Stored values:
- `production`
- `distribution`

## Tables

### movie
Prisma model: `Movie`

Columns:
- `id` bigint primary key generated by default as identity
- `title` text not null
- `release_year` integer not null
- `release_date` date null
- `film_type` text null
- `budget` bigint null
- `box_office` bigint null
- `created_at` timestamptz not null default now()
- `updated_at` timestamptz not null default now()

Indexes:
- index on `release_year`
- index on `film_type`

Prisma field mapping:
- `releaseYear` @map("release_year")
- `releaseDate` @map("release_date")
- `filmType` @map("film_type")
- `boxOffice` @map("box_office")
- `createdAt` @map("created_at")
- `updatedAt` @map("updated_at")

Relations:
- one-to-many with `MovieGenre`
- one-to-many with `MoviePerson`
- one-to-many with `MovieCompany`

### genre
Prisma model: `Genre`

Columns:
- `id` bigint primary key generated by default as identity
- `name` text not null

Constraints:
- unique(`name`)

Indexes:
- index on `name`

Relations:
- one-to-many with `MovieGenre`

### movie_genre
Prisma model: `MovieGenre`

Columns:
- `movie_id` bigint not null
- `genre_id` bigint not null

Primary key:
- composite primary key (`movie_id`, `genre_id`)

Foreign keys:
- `movie_id` references `movie(id)` on delete cascade
- `genre_id` references `genre(id)` on delete restrict

Indexes:
- index on `genre_id`

Prisma field mapping:
- `movieId` @map("movie_id")
- `genreId` @map("genre_id")

Relations:
- many-to-one with `Movie`
- many-to-one with `Genre`
